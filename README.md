# TASemWeb

A template for a website built on semantic web technologies.

## Features

- **HTML5 Semantic Elements**: Uses proper semantic HTML structure.
- **RDFa Annotations**: Includes RDFa attributes for structured data markup.
- **JSON-LD**: Embedded JSON-LD script for machine-readable metadata using Schema.org vocabulary.
- **Schema.org Integration**: Demonstrates use of Schema.org for web content markup.
- **Responsive Design**: Basic CSS for a clean, responsive layout.
- **JavaScript Demo**: Simple scripts to demonstrate data extraction from semantic markup.

## Files

- `index.html`: Main HTML file with semantic web markup.
- `style.css`: Basic styling for the website.
- `script.js`: JavaScript for interactivity and semantic data handling.

## Getting Started

1. Open `index.html` in a web browser to view the template.
2. Customize the content, structured data, and styling as needed.
3. Add more semantic annotations using RDFa or additional JSON-LD objects.

## Semantic Web Concepts Demonstrated

- **RDFa**: Resource Description Framework in Attributes for embedding structured data in HTML.
- **JSON-LD**: JavaScript Object Notation for Linking Data for standalone metadata.
- **Schema.org**: Vocabulary for structured data markup.
- **Microdata**: HTML5 microdata attributes (can be added as an alternative to RDFa).

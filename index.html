Welcome to the Semantic Web

This template demonstrates how to build websites using semantic web technologies. The semantic web allows data to be shared and reused across applications, enterprises, and communities.

What is the Semantic Web?

The Semantic Web is an extension of the World Wide Web through standards by the World Wide Web Consortium (W3C). The standards promote common data formats and exchange protocols on the Web, fundamentally the machine-readable information.

About This Template

This template includes:

  • HTML5 semantic elements
  • RDFa annotations for structured data
  • JSON-LD for machine-readable metadata
  • Schema.org vocabulary usage
